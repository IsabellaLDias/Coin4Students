## Histórias de Usuário - Coin4Students

### Aluno

- **US01** - Como aluno, quero me cadastrar no sistema para poder participar do programa de moedas.  

- **US02** - Como aluno, quero fazer login para acessar minha conta com segurança.  

- **US03** - Como aluno, quero visualizar meu saldo de moedas para saber quantas moedas possuo.  

- **US04** - Como aluno, quero consultar meu extrato para acompanhar minhas transações.  

- **US05** - Como aluno, quero receber notificações quando ganhar moedas para ficar informado sobre recompensas.  

- **US06** - Como aluno, quero visualizar as vantagens disponíveis para decidir como utilizar minhas moedas.  

- **US07** - Como aluno, quero resgatar uma vantagem usando minhas moedas para obter benefícios.  

- **US08** - Como aluno, quero receber um cupom por e-mail ao resgatar uma vantagem para utilizá-lo posteriormente.  


### Professor

- **US09** - Como professor, quero fazer login para acessar o sistema.  

- **US10** - Como professor, quero visualizar meu saldo de moedas para controlar a quantidade disponível.  

- **US11** - Como professor, quero enviar moedas para alunos para reconhecer seu desempenho acadêmico.  

- **US12** - Como professor, quero adicionar uma mensagem ao enviar moedas para justificar a recompensa.  

- **US13** - Como professor, quero consultar meu extrato para acompanhar as moedas distribuídas.  


### Empresa Parceira

- **US14** - Como empresa, quero me cadastrar no sistema para oferecer vantagens aos alunos.  

- **US15** - Como empresa, quero cadastrar vantagens com descrição, custo e imagem para disponibilizá-las no sistema.  

- **US16** - Como empresa, quero ser notificada quando uma vantagem for resgatada para validar a troca.  


### Sistema

- **US17** - Como usuário, quero autenticar minhas credenciais para acessar o sistema com segurança.  

- **US18** - Como usuário, quero receber e-mails automáticos para ser notificado sobre eventos importantes.  

- **US19** - Como usuário, quero receber um código de cupom ao realizar um resgate para validar a troca.
